# 9ITHON 2025

 구름톤 유니브 **경인/경기북부 지부** 해커톤 **9ITHON** 참가팀의 프로젝트 저장소입니다.  

## ☁️ 목적
**사회적 문제 해결**을 위한 실전형 IT 프로젝트 개발

## ☁️ 해커톤 개요

- **기간**: 2025년 7월 12일(토) ~ 7월 13일(일)
- **장소**: 경기 성남시 판교로 242 PDC A동 9층 구름스퀘어
- **참가자**: 전국 11개 대학, 약 60명
- **주최**: 구름톤 유니브 경인/경기북부 지부

## ☁️ 팀 저장소 네이밍 규칙

- `{팀명}-{프로젝트명}`
- 예: `TeamCloud-9roormMaker`

## ☁️ 추천 브랜치 전략

- `main`: 배포용
- `dev`: 통합 개발 브랜치
- `feat/{기능명}`: 기능 단위 개발 브랜치
- `fix/{버그설명}`: 버그 수정 브랜치
